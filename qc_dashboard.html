QC Checklist: MBES vs AUV Point Cloud vs SBP Position Comparison

Use this file to check whether an object's position agrees across three data sources: Multibeam Echosounder (MBES), AUV point cloud, and Sub-Bottom Profiler (SBP).

Sheets in this file

SheetPurpose
QC_M BES_VesselMBES survey checks for a crewed vessel or USV: GNSS/motion, calibration, SVP/tide, data acquisition.
QC_M BES_AUVMBES survey checks for an AUV: INS/DVL navigation, drift budget, calibration, mission planning, post-processing.
QC_SBP_VesselSBP survey checks for a crewed vessel or USV: positioning, transducer setup, signal/penetration, sediment velocity.
QC_SBP_AUVSBP survey checks for an AUV: navigation, sensor offset/altitude control, signal/penetration, post-processing.
QC_SSS_TowedSide scan sonar checks for a towed fish: layback/tow configuration, sonar settings, coverage, data quality.

Cross-comparison QC and calculations should be completed once you are comparing positions between data sources. Sheets include QC_Cross_Comparison, Position_Comparison and Error_Budget.

Key assumptions

Enter uncertainty values (Uh, Uv) at 95% confidence. If you only have a 1-sigma value, multiply by k first (default k = 1.96). Match rule: two positions are considered a match when the measured distance (horizontal or vertical) is no larger than the combined uncertainty: combined = SQRT(U1^2 + U2^2).

Uncertainty targets follow IHO S-44 (6th edition) by survey Order (Special / 1a / 1b / 2). If position errors are not equal in all directions, use a Mahalanobis distance / chi-square test instead of a simple circle.
